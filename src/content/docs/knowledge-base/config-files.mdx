> Introduced in [**v1.0**](/changelog/announcing-v1)

Config files allow you to create any arbitrary files within your services 
and to overwrite any arbitrary files within your services.

![Config file content](/images/tuto-bun-fn-config-file.png)

- You need specify a **mount path** for where the file should be located
- You can choose a **language** for syntax highlighting in the UI
- And you can defined a **name** to show in the UI
