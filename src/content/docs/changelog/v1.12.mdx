    <img className="!hidden dark:!block" src={`${ASSETS_SERVER_DOMAIN}/images/new-version-modal-dark.png`} alt="project environments page" />

2. via the shell:
    ```shell
    # assuming you are at /var/www/zaneops
    curl https://cdn.zaneops.dev/makefile > Makefile
    make setup
    make deploy
    ```
